Estandares web

El World Wide Web Consortium (W3C) es una organización internacional que desarrolla estándares para la World Wide Web. Tiene como objetivo garantizar que la web crezca de manera coherente y que sus estandares sean abiertos y accesibles para todos.

  • Scalable Vector Graphics (SVG) es un formato de imagen basado en XML que permite crear gráficos que se pueden escalar a cualquier tamaño sin perder calidad.

  • Web Open Font Format (WOFF) es un formato de fuente comprimida que permite a los diseñadores web utilizar fuentes personalizadas mientras se asegura una carga rápida y eficiente.

  • eXtensible Markup Language (XML) es un formato de archivo basado en texto utilizado para almacenar y transportar datos. Permite definir formatos propios de documento y datos.

  • Web Real-Time Communication (WebRTC) es un conjunto de tecnologías estándar que permite la comunicación de audio, video y datos en tiempo real entre navegadore y dispositivos.
