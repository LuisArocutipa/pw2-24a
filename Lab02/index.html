¡Bienvenido a mi página web!

Esta página web es creada como un proyecto para el curso de Programación Web 2 de la Escuela Profesional de Ingeniería de Sistemas de la Universidad Nacional de San Agustín.

El proyecto consta de crear un página web en la cual se mostrarán diferentes temas acerca de mí y la ingeniería de software, utilizando utilizando HTML y CSS, y desplegando el sitio en un servidor Docker.

Dentro del contenido que podrá encontrar en la página está: Descripción del autor, mis hobbies, una galería de fotos, información sobre la ingeniería de software, algunos estándares web y un formulario para contactarme.
